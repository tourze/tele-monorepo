# 干系人沟通技能

## 适用场景

- 项目跨部门协作，需要明确沟通责任、节奏、渠道。
- 需求、风险、变更需要及时同步与确认。
- 记录会议纪要、决定、行动项，建立追踪机制。

## 前置准备

- 识别干系人：角色、目标、关注点、影响力、沟通偏好。
- 准备模板：会议纪要、周报、决策记录、行动项表。
- 选择沟通渠道（邮件、IM、会议、公告板）。

## 操作步骤

1. **沟通矩阵**
   - 使用 RACI 或 SIPOC 定义责任（Responsible、Accountable、Consulted、Informed）。
   - 构建 Mermaid 组织图或关系图展示沟通路径。
2. **节奏规划**
   - 制定沟通日历：每日站会、周同步、里程碑评审、月度总结。
   - 为关键事件设置提醒（发布、风险、审批）。
3. **模板与记录**
   - 会议纪要模板：时间、参与者、讨论要点、决策、行动项、责任人、截止时间。
   - 周报/状态报告：里程碑进展、风险、阻塞、下一步计划。
   - 发布/变更通知模板：背景、影响、操作、回滚、联系方式。
4. **行动项追踪**
   - 使用工具（Issue Tracker、看板、表格）跟踪责任与进度。
   - 定期回顾未完成项，升级重要事项。
5. **决策可追溯**
   - 对重大决策生成哈希或编号，记录在知识库。
   - 保留附件、截图、投票结果。

## 质量校验

- 沟通矩阵覆盖所有关键干系人。
- 会议纪要、周报模板使用统一格式，版本受控。
- 行动项追踪有 Owner、截止时间、状态。
- 决策记录可快速检索、验证。

## 失败与回滚

- 沟通遗漏导致问题：制定补救沟通计划，更新矩阵。
- 信息不一致：同步纠正，建立单一事实源。
- 纪要缺失：补写并标注延迟原因，未来加强实时记录。

## 交付物

- 干系人沟通矩阵、关系图。
- 沟通节奏表、会议纪要、周报模板。
- 行动项追踪表、决策记录档案。
