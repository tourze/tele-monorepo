# 画布视觉设计技能

## 适用场景
- 用户需要原创海报、展板、静态插画等高视觉品质的定制作品。
- 需要以“设计哲学 + 视觉呈现”两段式交付，便于反复扩展系列作品。
- 要求作品保持大师级工艺感、独特风格并符合版权与品牌约束。

## 前置准备
- 收集用户需求：主题、情绪、色调、用途、输出规格（尺寸、分辨率、文件格式）。
- 明确是否引用自有品牌元素（Logo、字体、配色），确保授权可用。
- 建立素材整理区：`references/` 存放灵感板，`assets/` 存放可复用图形、纹理。

## 操作步骤
1. **构建设计哲学**
   - 为本次作品命名一个视觉运动（1–2 个中文/英文词，如「霓虹牧歌」）。
   - 编写 4–6 段宣言，覆盖空间/形态、色彩/材质、节奏/比例、视觉层级等维度。
   - 多次强调“匠心”、“精雕细琢”等关键词，传达作品需经过大量手工打磨的质感。
2. **建立视觉策略**
   - 根据宣言列出风格要素：主色板、辅助色、排版策略、图形语言、质感处理方法。
   - 制定留白与信息层级规划，确保文字仅作为点缀，视觉语言为核心。
   - 若需系列化输出，提前规划后续页面的差异化主题。
3. **画布创作与迭代**
   - 选择工具（Figma、Affinity、Illustrator 或生成式画布）进行创作，按 90% 视觉 + 10% 文本的比例布局。
   - 通过网格、对角线或黄金比例控制节奏，加入纹理、阴影、颗粒感提升工艺感。
   - 每次关键调整需保留版本快照，方便回滚与对比。
4. **输出与自检**
   - 导出 PNG（透明/不透明）或 PDF（向量/高分辨率）文件，命名包含风格与版本号。
   - 生成预览 Markdown，嵌入设计哲学与导出图像，便于团队审阅。

## 质量校验
- 检查分辨率与色彩模式是否符合交付要求（印刷场景使用 CMYK 或提供转换说明）。
- 对齐品牌/版权要求：字体、素材来源可追溯，避免使用未授权元素。
- 视觉检查：保证对比度、留白、对称/平衡均符合宣言描述；必要时请他人复审。
- 输出文件无多余空白页、字体缺失、色偏。

## 交付物
- 设计哲学宣言（Markdown），包含风格命名、核心原则、应用建议。
- 最终作品文件（PNG/PDF），如需多页作品则按顺序编号。
- 版本记录与后续扩展建议（例如系列下一幅应如何变化）。

## 回滚与演进
- 如视觉方向与用户预期偏差大，可回退至宣言阶段重新对焦，再重新创作。
- 建议建立风格档案库，记录配色、组件、纹理，以便后续快速扩展同一视觉运动。
